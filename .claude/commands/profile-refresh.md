# Команда: profile-refresh

Проведи **Quick Refresh** сквозного профиля (`ms-profile.md`) по навыку **ms-participant-profile** — режим 2 (конец модуля).

## Входные данные

Контекст (какой модуль только что закрыт / какой HW-Protocol): $ARGUMENTS

## Методология

**Канон:** [.cursor/skills/ms-participant-profile/SKILL.md](../../.cursor/skills/ms-participant-profile/SKILL.md), режим 2.

## Порядок выполнения

1. Прочитай текущий `ms-profile.md` по `MS_PROFILE_PATH`.
2. Загрузи HW-Protocol модуля + daily-notes этого модуля.
3. Спроси: «3 главных сдвига? Что неожиданно? Что подтвердилось?».
4. По 4–6 ключевым секциям профиля (EQ, ресурсы, уязвимости, скрытые мотивы, цели, защиты) — один короткий вопрос «изменилось ли?».
5. Покажи diff.
6. После подтверждения — инкрементируй версию, сохрани старую копию в `ms-profile.history/`, запиши в daily 2–3 строки.

## Итог

Обновлённый `ms-profile.md vN`, запись в daily.
