# Быстрый старт: присоединяюсь к курсу

> **Время:** 20 минут до полной готовности
> **Для кого:** вы не были на KickStart, присоединились позже, или ещё не настроили среду для работы с AI

Не переживайте — курс устроен так, что можно включиться с любого модуля. Каждый модуль самостоятельный, навыки накапливаются.

---

## Шаг 1: Создать AI-проект (5 мин)

**Вариант А — Claude.ai (самый быстрый):**
1. Откройте [claude.ai](https://claude.ai) → залогиньтесь
2. Нажмите **Projects** → **New Project**
3. Назовите: `AI Empowered Team`
4. В описании проекта напишите:

> Я руководитель команды. Ты — мой AI-наставник по лидерству. Помогаешь развивать навыки управления командой: коммуникация, процессы, стейкхолдеры. Работаешь в режиме экзоскелета — не делаешь за меня, а помогаешь думать.

5. Нажмите **Add content** → **Add text content** → вставьте текст навыка S3.00 (оркестратор стейкхолдеров) из файла `Module-3/Participant/504-M3-AI-Skill-Stakeholder-Orchestrator.md`

**Готово!** Можете начать диалог: «У меня ситуация со стейкхолдерами. Помоги разобраться.»

**Вариант Б — Claude Code / Cursor (для продвинутых):**
1. Клонируйте репозиторий: `git clone https://github.com/cless75/2026-03-Leadership-AI.git`
2. Откройте папку как проект в Claude Code или Cursor
3. Все навыки уже доступны в `BoK/Skills/`

---

## Шаг 2: Заполнить профиль (5 мин)

Скопируйте шаблон профиля и заполните о себе:
- Файл: `Student-AI-Pack/templates/participant-profile.template.md` → сохраните как `my-profile.md`
- Минимум: имя/псевдоним, роль, размер команды, опыт с AI

Профиль помогает AI давать релевантные подсказки под ваш контекст.

Курс идёт 4 модуля. Вот краткая карта — что пропустили и где найти материалы:
