# 2026-03-Leadership-AI — мастерская 504 (AI Empowered Team)

Автономный Obsidian-vault и студенческий AI-пакет для курса **504**. Состав: навыки в `.cursor/skills/`, команды Claude Code в `.claude/commands/`, материалы участника `Module-1/`, `Module-2/`, каталог [[Student-AI-Pack/README|Student-AI-Pack]].

## Быстрый старт

```bash
git clone https://github.com/cless75/2026-03-Leadership-AI.git
cd 2026-03-Leadership-AI
```

Откройте папку репозитория в **Cursor** или **Claude Code** как корень проекта.

Далее: [[Student-AI-Pack/INSTALL|INSTALL]], [[504-LEARNING-JOURNEY|учебный процесс]], [[504-HOMEWORK-GUIDE|домашние задания]].
