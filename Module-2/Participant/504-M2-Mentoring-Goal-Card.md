## Моя цель по Хоукинз #3

### Шкала 0–4

| Уровень | Ориентиры |
|---------|-----------|
| **0** | Процессы не формализованы; работа «как получится»; делегирую только когда не успеваю |
| **1** | Есть базовые ритмы, но нерегулярные; делегирование ситуативное; ритм на мне |
| **2** | Регулярные встречи работают; понятные зоны ответственности; замечаю паттерны |
| **3** | Процессы адаптируются под задачу; команда сама поддерживает ритм; делегирую осознанно (D1-D4) |
| **4** | Продуктивность (Performing): ритм саморегулируется; я — Growth/Transform, не Run; процессы эволюционируют |

### Кластеры мыслесессии 2026-04-07 (5 новых концепций)

| Кластер | Что делаю | Канон / опора |
|---------|-----------|----------------|
| **Оргтипология** (3 формы коллектива) | Определяю, какая форма (артель / рабочая группа / команда) обслуживает каждый шаг моей цепочки; ищу несоответствие «форма ↔ тип задачи» | [[Org-Forms-Artel-Group-Team]] |
| **AI-микрокоманда** | Вокруг 1 точки расширения собираю 3 роли (исполнитель + AI-настройщик + интегратор); проектирую, кто играет какую роль | [[AI-Microteam-Pattern]] |
| **Employee Journey RGT** | Разбираю, кто из людей цепочки в Run / Growth / Transform; проверяю, не застрял(а) ли я сам(а) в Run на ключевом шаге («золотая клетка») | [[Employee-Journey-RGT]] |
| **AI-делегирование (Бланшар × AI)** | Для каждой делегируемой AI-задачи определяю стиль S1–S4; явно отмечаю D4-задачи, которые НЕ нужно делегировать AI | [[SLII-Blanchard]] |
| **Экзоскелет vs коляска** | Для каждой выбранной точки расширения отвечаю: усиливает мой навык или замещает его; где риск деградации | [[Exoskeleton-vs-Wheelchair]] |
