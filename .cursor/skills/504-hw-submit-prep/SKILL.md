# Навык: подготовка сдачи ДЗ 504 (504-hw-submit-prep)

**Чеклист:** [[Student-AI-Pack/templates/hw-submission-checklist|hw-submission-checklist.md]]  
**Инструкции HW:** `Week-N/Participant/504-WN-HW-Instructions.md` (в этом vault есть W1 и W2).

## Когда применять

Перед отправкой ДЗ; участник указал неделю **N** (1–4).

## Процесс

1. Уточнить **неделю N**. Загрузить `Week-N/Participant/504-WN-HW-Instructions.md` из vault; для W3–W4, если файла нет — опереться на чеклист и обзор недели.
2. Пройти пункты [[Student-AI-Pack/templates/hw-submission-checklist|hw-submission-checklist]] с пользователем (да/нет / что заполнить).
3. Собрать **единый черновик markdown**: Job Story, сценарий AI/без AI, протокол по шаблону из инструкции недели, рефлексия (экзоскелет/коляска при W1).
4. Напомнить про **два канала** (мини-статус + полный протокол), если это указано в инструкции недели.

## Выход

- Готовый текст для копирования в Vault или экспорта.
- Список открытых пунктов, если чеклист не закрыт.
