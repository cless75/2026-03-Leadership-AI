# BoK — Body of Knowledge (База знаний курса)

**Курс:** AI Empowered MLE (Management, Leadership & Entrepreneurship)
**Рамка:** [[../CLAUDE|MLE Framework]]

---

## Структура

```
BoK/
  Principles/   — принципы (Connect Before Redirect, Экзоскелет vs Коляска, ...)
  Concepts/     — концепции (MLE Framework, Хоукинз, VxM, MWe, ...)
  Skills/       — навыки S0.xx - S4.xx
```

### Principles (Принципы)
Базовые правила, которые работают всегда. Не привязаны к конкретной неделе.

### Concepts (Концепции)
Модели и фреймворки. Каждая концепция имеет: определение, ключевую теорию, границы применимости, связи.

### Skills (Навыки)
AI-навыки курса. Нумерация: S{блок}.{NN}
- S0.xx — сквозные / личные компетенции
- S1.xx — Блок 1: Коммуникация (M1)
- S2.xx — Блок 2: Процессы (M2)
- S3.xx — Блок 3: Стейкхолдеры-структура (M3)
- S4.xx — Блок 4: Динамика + Миссия (M4)
- .00 = навык-оркестратор области

## Как использовать

1. **Загрузите BoK/ в Claude Project** — AI получит контекст всего курса
2. **Скажите «учи меня»** — AI проведёт по программе
3. **Или выберите конкретный навык** — S{блок}.{NN} по вашей задаче

## Принцип Сю-Ха-Ри

- **Сю (Следуй):** используйте навыки строго по промптам
- **Ха (Отступай):** адаптируйте промпты под свой контекст
- **Ри (За рамки):** создавайте свои навыки
