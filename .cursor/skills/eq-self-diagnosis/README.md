# eq-self-diagnosis — самодиагностика EQ (публичный навык)

Навык для Cursor / Claude Code: протокол **эмоционального интеллекта по модели Гоулмана** — слепая мини-диагностика (кейс + пять вопросов 1–5), три ловушки самооценки, «паутинка», одна точка фокуса; мини-кейсы; сценарии и наставник по компоненту; после диагностики — справочник концептов и лексики эмоций; протоколы **T1** (проверка гипотезы) и **T2** (разбор ошибок).

## Установка

Скопируйте папку `eq-self-diagnosis` в `.cursor/skills/` вашего проекта (или подключите навык в среде, где поддерживаются Agent Skills).

Команда (при наличии `.claude/commands`): `.claude/commands/eq-self-diagnosis.md` — скопируйте в свой репозиторий и при необходимости поправьте относительные пути к сценарию 218.

## Канон формулировок (опционально)

Полный текст **кейса 0** и вопросов совпадает со сценарием вебинара ДОД 504 (тема 218). Если в репозитории есть материалы курса, см. `2026/39/218/218-Сценарий-вебинар-ДОД-504.md`.
