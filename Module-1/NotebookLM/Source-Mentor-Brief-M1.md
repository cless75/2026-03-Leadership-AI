# Source — Mentor Brief M1

## Мета ведущего

- Удержать практический формат (не лекционный повтор KickStart).
- Провести участников через подготовку реального сложного разговора.
- Показать AI как экзоскелет для роста навыка коммуникации.

## Обязательные акценты

1. Не более 20 минут теории.
2. Обязательные breakout-пары.
3. Ролевое демо AI-наставника (не технический обзор).
4. Финал через "Три вещи для запоминания".

## "Три вещи для запоминания"

1. Два уровня обучения: модель + навык работы с AI.
2. Экзоскелет vs Коляска: AI должен развивать навык.
3. Результат недели - проведенный реальный разговор.

## Риски и митигации

| Риск | Что делать |
|---|---|
| Участники не сделали PreClass | 2 минуты на быстрый вход в начале сессии |
| Неравные пары по уровню | Свести пары "опытный + начинающий" |
| AI "делает за участника" | Явно задать роль наставника и вопросный режим |
| Перегрузка теорией | Таймер на блоки 1-4, переход в практику без сдвига |
