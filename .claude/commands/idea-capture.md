# Команда: Фиксация идеи (idea-capture)

**Навык:** [idea-capture](.cursor/skills/idea-capture/SKILL.md)

Быстрая фиксация идеи: запись в **ежедневную заметку** — секция `## Входящие дня`.

**Естественные триггеры (как в SKILL):** сообщение с **начала** с **«Идея»** + `:` / тире / пробел и текст; либо фраза **«зафиксируй идею»** (и аналоги).

## Входные данные (опционально)

- **$ARGUMENTS** — необязательный параметр:
  - текст идеи — сразу использовать пр�� записи;
  - `daily` / `ежедневная` — записать в ежедневную заметку.
- Если указан только текст идеи — записать в **ежедневную заметку** на сегодня (`YYYY/01-Daily/YYYY-MM-DD.md` → `## Входящие дня`). Если текста нет — о��ин вопрос: что зафиксировать.

## Обязательные источники (прочитать в начале)

1. **Навык:** `.cursor/skills/idea-capture/SKILL.md` — протокол, место записи.
2. **Справочник:** `.cursor/skills/idea-capture/reference.md` — пути, формат.

## Режим выполнения

- Один шаг за ответ.
- Запись в **`YYYY/01-Daily/YYYY-MM-DD.md`**, секция **`## Входящие дня`** (или отдельная заметка по шагу 1в SKILL — большая идея).
- Формат строки: `- [ ] {текст} → {источник} ⏰ HH:MM`.
