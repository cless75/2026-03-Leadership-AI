## Служебные навыки (в репозитории vault)

После сборки релиза они лежат в `.cursor/skills/`:

- **course-goals-intake** — провести опрос по шаблону целей, сохранить итог в `goals-504.md`.
- **participant-profile-load** — прочитать ваш заполненный профиль `.md` и выдать структурированное резюме для контекста.
- **504-hw-submit-prep** — собрать черновик сдачи ДЗ модули N по чеклисту.
- **stai-problem-solving** — полный pipeline системного решения проблемы по методологии курса **STAI** (514): формулирование, диагностика (5 почему, айсберг, CLD, архетипы), портфель гипотез, опционально реализация. Триггеры: «решить проблему», «системный анализ», «pipeline проблемы», «точка рычага». Команда в Claude Code: `.claude/commands/solve-problem.md`.

## Три способа получить пакет

1. **Архив (Google Drive и др.):** автор курса собирает дерево по MANIFEST → ZIP → вы скачиваете и открываете корень в Claude Code / Cursor. Подробно: [[MS-Courses/Course-Design/Course-Student-AI-Pack-Guide|Course-Student-AI-Pack-Guide]].
2. **Claude.ai:** загрузите папки навыков из MANIFEST как отдельные Skills (каждая с `SKILL.md` в корне).
3. **GitHub:** репозиторий для участников 504 — [cless75/2026-03-Leadership-AI](https://github.com/cless75/2026-03-Leadership-AI): клонировать и открыть корень как проект в Claude Code / Cursor. Состав и шаги — [[INSTALL|INSTALL]], [[MANIFEST|MANIFEST]].
