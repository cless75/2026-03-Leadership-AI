# Чеклист перед отправкой ДЗ (504)

Используйте с навыком **504-hw-submit-prep** или вручную перед сдачей в Telegram / Vault.

## Общие поля

- [ ] Указана **неделя N** (1–4)
- [ ] Есть **Job Story** или эквивалент из инструкции HW недели
- [ ] Отмечен сценарий: **с AI** / **без AI**
- [ ] Есть **протокол** (разговор / рефлексия) по шаблону из [[504-M1-HW-Instructions|HW Instructions]] или аналога M2–M4
- [ ] Убраны **ПДн** и идентификаторы третьих лиц (или получено согласие на учебный контекст)
- [ ] Самодиагностика / шкалы из PreClass учтены, если требовались неделей

## Связь с развитием навыков

- [ ] Назван **навык недели** (из библиотеки 504) и **слой AI** (D/U/N), если применимо
- [ ] Есть **рефлексия**: экзоскелет vs коляска (или аналог из материалов недели)
- [ ] **Карточка цели** модули согласована с фактическим опытом (если уже заполняли)

## Каналы

- [ ] Мини-статус в **Telegram** (если требуется когортой)
- [ ] Полный протокол в **Vault / Claude Project** (если это ваш канон)
