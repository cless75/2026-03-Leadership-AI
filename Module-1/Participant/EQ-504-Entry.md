# EQ на мастерской 504 — вход для участника

Модуль 1 опирается на **коммуникацию** и осознанную работу с собой и собеседником. Отдельный поток — **эмоциональный интеллект (EQ)** по модели **Гоулмана**: не оценка «уровня личности», а **практика** в реальных ситуациях.

## Зачем это вам

- Различать эмоции, регулировать реакции, точнее понимать контекст разговора (см. также [[MS-Courses/Concepts/Hawkins-Team-Competencies|Hawkins-Team-Competencies]]).
- Связка с AI: не делегировать «человеческую» часть диалога модели без рефлексии ([[MS-Courses/Concepts/Exoskeleton-vs-Wheelchair|Экзоскелет vs коляска]]).

## Как работать в AI-среде (Claude / Cursor / Claude Code)

В студенческом дистрибутиве (ZIP / GitHub) в корне проекта лежит папка **`.cursor/skills/eq-self-diagnosis/`** — её нужно сохранить при распаковке.

1. Откройте **корень** распакованной папки как проект в Cursor или Claude Code.
2. В чате используйте триггеры, например: **«самодиагностика EQ»**, **«мини-кейс EQ»**, **«наставник по компоненту EQ»**, **«разбор по Гоулману»**.

Канон навыка в полном vault: [[.cursor/skills/eq-self-diagnosis/SKILL|eq-self-diagnosis]] (протокол, границы: не терапия и не HR-оценка).
