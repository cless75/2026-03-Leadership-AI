# Команда: burnout-check

Запусти **самодиагностику выгорания** по навыку **burnout-self-diagnosis** по шкалам Maslach + физиологические маркеры + JD-R. Три режима по умолчанию — Quick Check (5–7 мин, 12 вопросов). Вывод — одна из 4 зон: 🟢 норма / 🟡 дефицит / 🟠 выгорание в развитии / 🔴 острое выгорание.

## Входные данные

Контекст (если участник просит full / recheck или у него есть предыдущий результат): $ARGUMENTS

## Методология

**Канон навыка:** [.cursor/skills/burnout-self-diagnosis/SKILL.md](../../.cursor/skills/burnout-self-diagnosis/SKILL.md)
**Вопросы и расчёт зон:** [.cursor/skills/burnout-self-diagnosis/reference.md](../../.cursor/skills/burnout-self-diagnosis/reference.md)
**Шаблон отчёта:** [.cursor/skills/burnout-self-diagnosis/templates/burnout-report.template.md](../../.cursor/skills/burnout-self-diagnosis/templates/burnout-report.template.md)

## Порядок выполнения

1. Прочитай SKILL.md и выбери режим: Quick (по умолчанию) / Full / Re-check.
2. Прогрев: 1 вопрос «как ты сейчас — одной фразой?».
3. Задай 12 (Quick) или 22 (Full) вопроса из reference. Можно блоком, если участник просит «давай сразу».
4. Подсчитай баллы по 4 шкалам (3 Maslach + физиология).
5. Определи зону.
6. Выведи короткий отчёт: зона, шкалы в высокой зоне, 3 конкретные рекомендации под зону.
7. При **🔴** — обязательный письменный блок с рекомендацией к психотерапевту / врачу (не «может быть», а «в течение 1–2 недель»), даже если участник скажет «я подумаю».
8. Предложи записать результат в `My-Notes/burnout-YYYY-MM-DD.md` (по шаблону) + обновить секцию 8 `ms-profile.md`.
9. Запланируй re-check в daily (дата через 7–30 дней в зависимости от зоны).

## Важно

- **Не клинический диагноз.** Операционный сигнал, не медицинская оценка.
- **Privacy.** Файл результата — только локально, не в git.
- **Без давления.** Участник может отказаться пройти diagnostic; отказ фиксируется нейтрально. Исключение — если в материалах есть прямые признаки кризиса (суицидальные мысли, тяжёлая бессонница, тяжёлая соматика) — письменная рекомендация к врачу обязательна независимо от прохождения.
