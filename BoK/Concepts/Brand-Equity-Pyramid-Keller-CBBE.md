# Пирамида капитала бренда (Brand Equity Pyramid / CBBE, Keller)

## Определение (одна строка)

> Капитал бренда (Customer-Based Brand Equity) выстраивается **снизу вверх** через 4 уровня (Identity → Meaning → Response → Resonance) и 6 building blocks. Без нижних уровней верхние не работают: невозможно построить лояльность (Resonance) без узнаваемости (Salience).

---

## Развёрнутое описание

### Четыре уровня и шесть building blocks

```
                    Resonance              ← Уровень 4: Relationships («Какие у нас отношения?»)
                  /     |     \
            Judgments   |   Feelings        ← Уровень 3: Response («Что я думаю/чувствую?»)
              /         |         \
        Performance     |     Imagery       ← Уровень 2: Meaning («Что это такое?»)
                  \     |     /
                    Salience                ← Уровень 1: Identity («Кто это?»)
```

(в Markdown без псевдографики — см. описание ниже)

| Уровень | Building block | Вопрос потребителя | Что измеряем |
|---------|----------------|--------------------|--------------|
| **1. Identity** | **Salience** | «Кто этот бренд?» | Узнаваемость, заметность, top-of-mind |
| **2. Meaning** | **Performance** | «Что он делает?» | Функциональные характеристики: качество, надёжность, эффективность, цена-ценность, сервис |
| **2. Meaning** | **Imagery** | «Что он значит?» | Символические/социальные ассоциации: history, profile of users, contexts of use, values |
| **3. Response** | **Judgments** | «Что я о нём думаю?» | Рациональные оценки: качество, доверие, превосходство, актуальность |
| **3. Response** | **Feelings** | «Что я чувствую?» | Эмоциональные реакции: тепло, веселье, волнение, безопасность, социальное одобрение, самоуважение |
| **4. Resonance** | **Resonance** | «Какие у нас отношения?» | Поведенческая лояльность, attitudinal attachment, sense of community, active engagement |

### Логика подъёма

Пирамида читается **снизу вверх** как путь зрелости отношений:

1. **Identity (Salience):** клиент сначала **узнаёт** бренд. Без узнаваемости ничего дальше не работает
2. **Meaning (Performance + Imagery):** клиент понимает **что делает** бренд (функция) и **что значит** (символ). Левая колонка — рациональная, правая — символическая
3. **Response (Judgments + Feelings):** клиент формирует **отклик**. Левая — рациональный (оценки), правая — эмоциональный (чувства)
4. **Resonance:** клиент строит **отношения** с брендом. Это вершина — самая зрелая стадия CBBE

### Что отличает CBBE от Brand Awareness

- Awareness — это **только** Salience (уровень 1)
- CBBE измеряет **всю пирамиду** — где бренд находится сейчас и куда движется
- Большинство брендов застревают на уровне 2 (Meaning): известны, понимают что делают, но не вызывают сильного отклика и не имеют лояльности

## Адаптация MS: CBBE × Стратегический цикл × RGT

### Маппинг уровней на стратегический цикл управления

Стратегический цикл MS (4 перспективы, 8 процессов, 2 петли) и CBBE — про разные масштабы (бренд vs. бизнес-операции), но логика подъёма зрелости совпадает:

| Уровень CBBE | Перспектива стр. цикла | Режим RGT | Что делать в команде |
|---------------|------------------------|-----------|----------------------|
| **1. Identity (Salience)** | Discovery / Marketing | Run | Базовая узнаваемость: имя, лого, основной нарратив. Контент-производство, охват |
| **2. Meaning (Performance + Imagery)** | Customer / Product | Run + Grow | Доказательство ценности (Performance) + построение символического слоя (Imagery): кейсы, отзывы, истории |
| **3. Response (Judgments + Feelings)** | Customer / Strategy | Grow | Активная работа с восприятием: отстройка от конкурентов, эмоциональная связка с миссией |
| **4. Resonance** | Strategy / Culture | Transform | Сообщество вокруг бренда, advocacy, со-создание ценности с клиентами |

### Инсайт адаптации

**Большинство стартапов и команд** делают типичную ошибку: переходят сразу с уровня 1 (есть имя) на уровень 4 (хотим лояльность), пропуская Meaning и Response.

В MS-курсах это переводится как:
- **«Run» брендовых коммуникаций** = уровни 1-2 (Salience + Performance)
- **«Grow» брендовых коммуникаций** = уровни 2-3 (Imagery + Judgments + Feelings)
- **«Transform» брендовых коммуникаций** = уровень 4 (Resonance, сообщество)

Каждый уровень требует **другого типа усилий**: Run = регулярная мощность, Grow = качественная отстройка, Transform = культурная работа с сообществом.

### Roadmap движения вверх

Для команды из защиты M4 (трек C):
- Если **сейчас уровень 1** (только узнают) → следующий шаг: построить Performance (что бренд реально делает) + один Imagery-нарратив
- Если **сейчас уровень 2** → следующий шаг: артикулировать дифференциацию для Judgments + 1-2 эмоциональных триггера для Feelings
- Если **сейчас уровень 3** → следующий шаг: создать ритуал/площадку для Resonance (комьюнити, регулярная встреча, общий артефакт)
- Между уровнями нельзя «прыгать»: попытка построить Resonance без сильного Meaning даёт фейковую лояльность, которая сыпется при первом давлении

## Логика модели

### Ядро (инварианты)

1. **Пирамида читается снизу вверх.** Невозможно построить верхний уровень без среднего и нижнего. Resonance без Meaning = фейковая лояльность
2. **Двойная структура (рациональное / символическое).** На уровнях 2 и 3 каждое building block разделено на левую (рациональную) и правую (символическую/эмоциональную) колонки. Сильный бренд развивает **обе** колонки
3. **CBBE — про восприятие клиента, не про намерение бренда.** Это «зеркало»: что **получает** клиент. Для измерения нужны исследования, не самооценка
4. **Resonance — поведенческий, не декларативный.** Лояльность измеряется повторными покупками, рекомендациями, активным engagement, а не словами «я люблю этот бренд»

### Взаимосвязи

- **[[Brand-Identity-Prism-Kapferer|BIP (Kapferer)]]:** парная модель. BIP даёт **состав** идентичности (sender's perspective), CBBE — **зрелость** воспринятого капитала (receiver's perspective). Вместе: BIP отвечает «что мы транслируем», CBBE — «что аудитория получила и насколько глубоко»
- **[[5-Mission-For-Developer]]:** миссия команды питает Imagery (символический слой) и Resonance (сообщество вокруг смысла)
- **[[Strategic-Cycle]]:** уровни CBBE накладываются на перспективы стратегического цикла (см. таблицу адаптации)
- **[[8-Ecosystem-Competition]]:** в экосистемной конкуренции Resonance критичен — брендовое сообщество = главный лок-ин

### Границы применимости

- CBBE разработан для **B2C-рынков** в первую очередь. В B2B-контексте уровень Feelings работает иначе (через личные отношения с менеджером, а не эмоциональную связь с брендом)
- Для **молодого бренда** (< 1 года) измерять Resonance бессмысленно — нет данных. Фокусироваться на Salience + Performance
- CBBE — **аналитическая** модель, не операционная. Не говорит «что делать» — говорит «где ты находишься». Для решения нужны другие инструменты (см. roadmap выше)
- Не работает для **закрытых рынков** (госзаказ, монополии) — там лояльность не строится через CBBE-механизмы
