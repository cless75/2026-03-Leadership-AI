## Промпт 1: Обзор теории модуля (основной, 10-12 мин)

```
Создай аудио-обзор модуля 1 курса AI Empowered MLE.

ВАЖНО: все названия фреймворков — на русском первым, английское в скобках при первом упоминании.

Тема: Коммуникация (область C1 — «Кто я и кто мне нужен»).

Структура обзора:
1. Позиция M1 в курсе: первый шаг контура идентификации (Identity Loop) C1→C5. Начинаем с себя — кто я, как я общаюсь, какие отношения мне нужны.
2. Ключевые концепции (по 1-2 мин каждая):
   - Сначала контакт, потом направление (Connect Before Redirect) — 3 уровня контакта — базовый принцип всей коммуникации
   - Модели обратной связи SBI (Ситуация-Поведение-Влияние) и BOFF (Поведение-Результат-Чувства-Будущее) — структура, не манипуляция
   - Ситуационное лидерство Бланшара (Situational Leadership) — готовность D1-D4 по задаче, не по человеку. Стиль лидера S1-S4
   - Ненасильственное общение (NVC) — 4 шага Розенберга + шаг 0 = Сначала контакт, потом направление
3. Три роли AI: Аналитик / Наставник / Коуч — какую роль вызывать для коммуникации
4. Экзоскелет и коляска (Exoskeleton vs Wheelchair) — AI усиливает навык, не подменяет суждение. Тест: «Если убрать AI — смогу ли я сам?»
5. Что делать на этом модуле: выбрать один навык (S1.01-S1.05), пройти один сценарий на своём кейсе

Тон: дружелюбный, практичный. Без академизма. Примеры — из реальной управленческой практики.
Длительность: 10-12 минут.
```

## Промпт 2: Подготовка к обзорной сессии (Pre-Review Q&A, 5-7 мин)

```
Создай аудио-подготовку к обзорной сессии (Review) по модулю 1.

ВАЖНО: названия фреймворков — на русском первым.

Разбери 3 реальных вопроса из мастерской M1:

1. «Почему AI не вызывает защитную реакцию, а живой человек — вызывает?» (Алексей)
   — Ответ: принцип «Сначала контакт, потом направление» (Connect Before Redirect) объясняет — AI не активирует лимбическую систему (нет контекста угрозы). Используйте AI как безопасную среду для репетиции.

2. «Профессиональные уколы больнее личных — как давать обратную связь эксперту?» (Андрей)
   — Ответ: для эксперта (уровень D4) контакт особенно важен — подтверждение компетенции перед обратной связью. Модель SBI без оценки компетенции: только поведение и влияние.

3. «Можно ли применять ненасильственное общение (NVC) без искренней эмпатии?» (Юлия)
   — Ответ: в рабочем контексте эмпатию заменяет профессиональное уважение к роли. «Сначала контакт, потом направление» работает без «тёплых чувств» — это последовательность, не настроение.

Заверши: что взять с собой на обзорную сессию (свой кейс + 1 вопрос).
Тон: как разговор с наставником, не лекция.
Длительность: 5-7 минут.
```
