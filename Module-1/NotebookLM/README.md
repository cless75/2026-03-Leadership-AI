# NotebookLM Package — 504 M1 (Коммуникация)

Полный локальный пакет для загрузки в NotebookLM по модуле 1.

## Что входит в пакет

1. `Lesson-Core.md` — ядро недели: цели, блоки, концепты.
2. `Source-PreClass-M1.md` — материал для участников до воркшопа.
3. `Source-RunOfShow-M1.md` — логика проведения живой сессии.
4. `Source-Mentor-Brief-M1.md` — ориентиры ведущего.
5. `Source-Actualized-Plan-M1.md` — актуализация после KickStart.
6. `Prompt-Pack-M1.md` — готовые промпты для NLM Studio.
7. `Source-Manifest-M1.md` — порядок загрузки и назначение источников.
8. `Upload-Checklist-M1.md` — быстрый чеклист перед запуском генераций.

## Рекомендуемый порядок работы в NotebookLM

1. Загрузить источники по порядку из `Source-Manifest-M1.md`.
2. Проверить, что в ноутбуке не больше 7 основных источников.
3. Запустить из `Prompt-Pack-M1.md`:
   - Study Guide (до воркшопа),
   - Quiz (входной),
   - Flashcards.
4. После сессии запустить:
   - Study Guide (после),
   - Quiz (post-session),
   - Fact-check.
