# AI-наставник по обучению (S0.00)

> **Роль AI:** Персональный наставник по программе курса. Ведёт вас по материалам, помогает разобраться в концепциях, готовит к воркшопам и менторингу.

> **Происхождение:** паттерн Алексея из когорты W2 — «загрузил все материалы в Claude, сказал учи меня». За 2 недели прошёл от нуля до активного пользователя.

---

## Как это работает

Загрузите материалы курса в Claude Project → AI становится вашим персональным наставником. Он знает программу, концепции, навыки и может вести вас по курсу в вашем темпе.

**Три режима:**
- **Учи меня** — AI ведёт по программе, объясняет концепции, задаёт вопросы для проверки понимания
- **Помоги с ДЗ** — AI помогает выполнить домашнее задание пошагово, подбирает нужный навык
- **Подготовь к сессии** — AI готовит вас к Review / Workshop / менторингу: ключевые концепции, что принести, на что обратить внимание

## Быстрый старт (5 минут)

### Шаг 1: Создайте проект в Claude
Откройте claude.ai → Projects → Create Project → назовите «504 MLE»

### Шаг 2: Загрузите материалы
Добавьте в проект файлы из репозитория курса (или из Telegram):
- Навыки текущей недели (S-файлы)
- PreClass текущей недели
- HW Instructions
- Любые другие материалы, которые у вас есть

### Шаг 3: Скажите «Учи меня»

```text
Ты — мой персональный наставник по программе курса AI Empowered MLE.

В проекте загружены материалы курса. 

Помоги мне:
1. Понять, где я сейчас в программе (какая неделя, какая тема)
2. Что я должен был сделать и что мог пропустить
3. Проведи меня по ключевым концепциям текущей недели
4. Помоги подготовиться к ближайшей сессии

Веди меня шаг за шагом. Задавай вопросы, чтобы проверить, понимаю ли я.
```

## Дополнительные промпты

### «Помоги с ДЗ»

```text
Мне нужно сделать домашнее задание недели [N].

Вот что требуется: [вставить из HW Instructions или описать своими словами]

Помоги мне:
1. Выбрать подходящий навык (S-файл)
2. Пройти по шагам
3. Проверить результат
4. Подготовить рефлексию

Я работаю с [моя ситуация / учебный кейс TechVision].
```

### «Подготовь к сессии»

```text
Завтра у меня [Review / Workshop / менторинг] по неделе [N].

Помоги мне подготовиться:
1. Какие ключевые концепции будут обсуждаться?
2. Что я должен принести / подготовить?
3. Какие вопросы мне стоит задать?
4. Есть ли что-то из прошлых недель, что я мог пропустить?
```

## Принцип: экзоскелет, не коляска

AI-наставник **усиливает** ваше обучение, но не подменяет его. Ключевая разница:

| Экзоскелет (правильно) | Коляска (неправильно) |
|---|---|
| AI объясняет концепцию → вы применяете к своему кейсу | AI делает ДЗ за вас |
| AI задаёт вопросы → вы формулируете ответы | AI пишет ответы, вы копируете |
| AI подсвечивает слепые зоны → вы решаете, что делать | AI решает за вас |
