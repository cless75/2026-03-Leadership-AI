# Установка пакета 504 для Claude Code / Cursor

## Вариант: клон с GitHub

Репозиторий для участников: [github.com/cless75/2026-03-Leadership-AI](https://github.com/cless75/2026-03-Leadership-AI).

```bash
git clone https://github.com/cless75/2026-03-Leadership-AI.git
```

Откройте папку клона как корень проекта в Cursor или Claude Code (внутри должны быть `.cursor/skills/` и `Student-AI-Pack/`).

## После скачивания архива

1. Распакуйте ZIP в папку без кириллицы в пути (рекомендация для инструментов).
2. Откройте **эту папку** как корень проекта:
   - **Claude Code:** открыть папку, внутри которой есть каталог `.cursor/skills/`.
   - **Cursor:** File → Open Folder → выбрать корень распаковки.
3. Убедитесь, что видны навыки: `.cursor/skills/` содержит подпапки с файлами `SKILL.md`.

## Профиль и личные файлы

- Скопируйте [[templates/participant-profile.template|participant-profile.template.md]] в `my-profile.md` (или другое имя) и заполните.
- Файлы `my-profile.md` и `goals-504.md` в `.gitignore` — не коммитьте личные данные в публичный репозиторий.
