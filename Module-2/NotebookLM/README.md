## Рекомендуемый порядок работы в NotebookLM

1. Загрузить источники по порядку из `Source-Manifest-M2.md` (нарративные **1–3** первыми)
2. Проверить, что в ноутбуке **не больше 9** источников
3. Запустить из `Prompt-Pack-M2.md`: Slide Deck Часть 1 + Часть 2
4. **NLM Slide Review** — 10 проверок (+ при необходимости покрытие value chain / S09)
5. Сгенерировать **пост-квиз** (промпт в Prompt-Pack) для выравнивания с ДЗ v2
6. Audio Overview пре/пост, Study Guide, практический чеклист — по чеклисту загрузки
