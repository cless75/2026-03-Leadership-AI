# Быстрый старт: присоединяюсь к курсу

> **Время:** 20 минут до полной готовности
> **Для кого:** вы не были на KickStart, присоединились позже, или ещё не настроили среду для работы с AI

Не переживайте — курс устроен так, что можно включиться с любого модуля. Каждый модуль самостоятельный, навыки накапливаются.

---

## Шаг 1: Создать AI-проект (5 мин)

**Вариант А — Claude.ai (самый быстрый):**
1. Откройте [claude.ai](https://claude.ai) → залогиньтесь
2. Нажмите **Projects** → **New Project**
3. Назовите: `AI Empowered Team`
4. В описании проекта напишите:

> Я руководитель команды. Ты — мой AI-наставник по лидерству. Помогаешь развивать навыки управления командой: коммуникация, процессы, стейкхолдеры. Работаешь в режиме экзоскелета — не делаешь за меня, а помогаешь думать.

5. Нажмите **Add content** → **Add text content** → вставьте текст навыка S3.00 (оркестратор стейкхолдеров) из файла `Module-3/Participant/504-M3-AI-Skill-Stakeholder-Orchestrator.md`

**Готово!** Можете начать диалог: «У меня ситуация со стейкхолдерами. Помоги разобраться.»

**Вариант Б — Claude Code / Cursor (для продвинутых):**
1. Клонируйте репозиторий: `git clone https://github.com/cless75/2026-03-Leadership-AI.git`
2. Откройте папку как проект в Claude Code или Cursor
3. Все навыки уже доступны в `BoK/Skills/`

Скопируйте шаблон профиля и заполните о себе:

Профиль помогает AI давать релевантные подсказки под ваш контекст.

## How-to: установка инструментов

Ниже — краткая инструкция по каждому инструменту. Ставьте только то, что нужно прямо сейчас. Можно работать только с Claude.ai и добавлять инструменты по мере необходимости.

### Claude.ai (обязательно, 5 мин)

1. Откройте [claude.ai](https://claude.ai)
2. Зарегистрируйтесь (email или Google-аккаунт)
3. Создайте проект: **Projects → New Project → «AI Empowered Team»**
4. Загрузите навык: **Add content → Add text content** → вставьте текст `504-M3-AI-Skill-Stakeholder-Orchestrator.md`

> **Проблемы с доступом?** Если Claude.ai недоступен в вашем регионе — используйте VPN. Бесплатный тариф работает, но с ограничениями по количеству сообщений. Подписка Claude Pro снимает лимиты.

### Obsidian (рекомендуется, 20 мин)

Obsidian — ваша библиотека навыков. За 4 модуля вы соберёте 16+ AI-скиллов, заметки с занятий, результаты диагностик.

1. **Скачайте:** [obsidian.md/download](https://obsidian.md/download) — Windows / macOS / Linux / iOS / Android
2. **Установите** и запустите
3. **Создайте vault:** при первом запуске выберите **Create new vault**
   - Название: `504-AI-Empowered-Team`
   - Расположение: любая папка без кириллицы в пути (например, `C:\Obs-Vaults\` или `~/Documents/`)
4. **Создайте первую заметку:** `Ctrl+N` → вставьте текст навыка → сохраните как `S3-00-Оркестратор.md`

> **Зачем Obsidian, а не просто файлы?** Obsidian связывает заметки через `[[wikilinks]]` — ваши навыки, модели и кейсы образуют сеть знаний. Через месяц это будет персональная база, которую не построить в Google Docs.

**Рекомендуемые настройки** (Settings ⚙️):
- Editor → Default editing mode → **Source mode**
- Files & Links → Use Wikilinks → **включено**
- Editor → Properties in document → **Visible**

### Claude Code (опционально, 30 мин)

Claude Code — AI в терминале, который видит ваши файлы и работает с контекстом.

1. **Установите Node.js:** [nodejs.org](https://nodejs.org/) → LTS-версия → установщик для вашей ОС
2. **Установите Claude Code:** откройте терминал → `npm install -g @anthropic-ai/claude-code`
3. **Запустите:** перейдите в папку vault → `claude` → авторизуйтесь
4. **Проверка:** напишите `помоги мне загрузить навык S3.00` — Claude Code найдёт файл и предложит действия

> Подробная инструкция: [[../Operations/504-g-Environment-Setup|Environment Setup]]

### Git + GitHub (опционально, 15 мин)

Если хотите получать обновления материалов курса автоматически:

1. **Установите Git:** [git-scm.com/downloads](https://git-scm.com/downloads)
2. **Клонируйте репозиторий:**
   ```
   git clone https://github.com/cless75/2026-03-Leadership-AI.git
   ```
3. **Откройте** клонированную папку как vault в Obsidian
4. **Обновление:** `git pull` в терминале — получаете новые материалы

> Не обязательно для работы на курсе. Альтернатива — скачать ZIP с GitHub.
