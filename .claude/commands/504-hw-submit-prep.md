# Команда: 504-hw-submit-prep

Подготовь **черновик сдачи домашнего задания** мастерской 504 по навыку **504-hw-submit-prep**: чеклист, соответствие инструкции недели N, единый markdown для отправки.

## Входные данные

Неделя N и контекст (опционально): $ARGUMENTS

Если неделя не указана — запроси **N** (1–4).

## Методология

**Канон навыка:** [.cursor/skills/504-hw-submit-prep/SKILL.md](.cursor/skills/504-hw-submit-prep/SKILL.md)

**Чеклист:** [Student-AI-Pack/templates/hw-submission-checklist.md](../../Student-AI-Pack/templates/hw-submission-checklist.md)

**Инструкция:** `Week-N/Participant/504-WN-HW-Instructions.md`

**Справочник процесса:** [504-HOMEWORK-GUIDE.md](../../504-HOMEWORK-GUIDE.md)

## Порядок выполнения

1. Прочитай `SKILL.md` и открой инструкцию HW для недели N.
2. Пройди чеклист с пользователем.
3. Собери единый черновик по шагам SKILL.
