# Module 2 — Процессы и сотрудничество (Хоукинз #3)

> **Обновление 2026-04-09:** воркшоп M2 (2026-04-07) прошёл как авторская мыслесессия с 5 новыми концепциями (AI-микрокоманда, 3 формы коллектива, Employee Journey RGT, Бланшар × AI-делегирование, экзоскелет vs коляска). Главное ДЗ перенесено на **построение цепочки создания ценности (value chain)** через навык-оркестратор **S09**. См. пакет «После воркшопа» ниже.

## Цель недели (v2)

Построить **карту цепочки создания ценности** своей команды (или вида деятельности) и найти 1–3 точки расширения с помощью AI; провести точечный эксперимент по узкому месту через один из навыков S05–S08.

## Результаты недели

- участник имеет **карту value chain** на 5–9 шагов с обозначенным bottleneck;
- участник имеет **1–3 точки расширения AI** с приоритизацией Impact × Effort и стилем Бланшара;
- участник явно отметил **минимум 1 антипаттерн** (где AI = коляска или «золотая клетка»);
- участник провёл **один точечный эксперимент** на узком месте (S05/S06/S07/S08).
