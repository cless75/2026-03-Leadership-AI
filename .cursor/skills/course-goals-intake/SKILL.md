# Навык: опрос целей на курс (course-goals-intake)

**Шаблон:** [[Student-AI-Pack/templates/goals-intake|goals-intake.md]]  
**Пакет курса:** [[Student-AI-Pack/README|Student-AI-Pack README]]

## Когда применять

Запросы участника или ментора: зафиксировать ожидания от мастерской 504 перед стартом или в первую неделю.

## Процесс

1. Прочитать шаблон `goals-intake.md` из пакета (если доступен в vault); иначе использовать структуру секций из шаблона по памяти навыка.
2. Вести диалог **по одному блоку за шаг** (ожидания → недели → AI-зрелость → ограничения → итог одной строкой).
3. В конце сформировать **готовый markdown** для сохранения пользователем как `goals-504.md` (или имя по договорённости).
4. Напомнить: не включать ПДн; при публичном обмене — обезличить контекст.

## Выход

- Один файл-итог в формате шаблона с заполненными ответами.
- Опционально: 3 bullet «для ментора» в конце.
