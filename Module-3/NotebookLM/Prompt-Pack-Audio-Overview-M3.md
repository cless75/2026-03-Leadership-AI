## Промпт 1: Обзор теории модуля (10-12 мин)

```
Создай аудио-обзор модуля 3 курса AI Empowered MLE.

ВАЖНО: все названия фреймворков — на русском первым, английское в скобках при первом упоминании.

Тема: Стейкхолдеры и внешняя среда (область C4 — «Кто вокруг нас и чего хотят»). Экватор курса — внешний радар.

Структура:
1. Позиция M3: контур идентификации (Identity Loop) C1 (кто я) → C3 (как мы работаем) → C4 (кто вокруг). Внешний радар: команда не существует в вакууме.
2. Ключевые концепции:
   - Матрица влияния и интереса (Power/Interest Matrix) — 4 квадранта стейкхолдеров: Ключевые партнёры (управлять плотно), Поддерживать удовлетворённость, Регулярно информировать, Наблюдать. Скрытые стейкхолдеры — неформальные лидеры, привратники (gate-keepers).
   - Коммуникация вверх (Manage Up) — Главный вывод первым (BLUF / Bottom Line Up Front), Записка по решению (Decision Memo): ситуация → варианты → рекомендация → запрос. Стиль руководителя.
   - Принципиальные переговоры (Fisher & Ury, «Переговоры без поражений» / Getting to Yes) — интересы, не позиции. Лучшая альтернатива (BATNA) и Зона возможного соглашения (ZOPA). Правила Раппопорта (Rapoport Rules).
   - Модель конфликтных стратегий Томаса-Килмана (Thomas-Kilmann / TKI) — 5 стратегий: Соперничество / Сотрудничество / Компромисс / Уклонение / Уступка.
   - Ассертивная позиция (Assertiveness) — экологичный внутренний настрой важнее экологичных методов. Эвристика 5 минут.
3. Принцип «Сначала контакт, потом направление» (Connect Before Redirect) работает и со стейкхолдерами (callback M1)
4. Навыки M3: S3.00 оркестратор + S3.01-S3.04 (аналитик / коуч / наставник)
5. Парная работа (breakout) на мастерской: строим карту стейкхолдеров в парах — впервые за курс!
6. Задание: составить список 5+ стейкхолдеров до мастерской

Тон: энергичный (экватор!). Акцент: вы наработали фундамент (C1 коммуникация, C3 процессы), теперь применяем наружу.
Длительность: 10-12 минут.
```

```
Создай аудио-подготовку к обзорной сессии (Review) по модулю 3.

ВАЖНО: названия фреймворков — на русском первым.

Контекст: это экватор курса. На обзорной сессии (пн) разбираем ДЗ M2 + закрываем долг «Три слоя AI-навыков» (Three Layers AI) + предпросмотр стейкхолдеров.

Разбери 3 темы:

1. «Экологичные методы при неэкологичном внутреннем настрое ≠ экологичная коммуникация» (из менторинга M2, Алексей)
   — Внутреннее состояние важнее техник. Если внутри «образ врага» — даже лучшие техники считываются как скрытая агрессия. Решение: попросить AI помочь найти ассертивную позицию (Assertiveness) перед разговором.

2. «Загрузи всё + скажи учи меня» — паттерн Алексея
   — Работающий подход к освоению курса: загрузить материалы в Claude Project, сказать «учи меня по программе». AI становится персональным наставником. Попробуйте до обзорной сессии!

3. Три слоя AI-навыков (Three Layers: D/U/N) — наконец разберём
   — Три слоя: Диагност (найти паттерн), Усилитель (улучшить формулировку), Наставник (разобрать опыт). На M3 используем все три: аналитик для карты, коуч для коммуникации вверх (Manage Up), наставник для переговоров.

Заверши: что взять на обзорную сессию (результат ДЗ M2 + список 5 стейкхолдеров + 1 вопрос).
Длительность: 5-7 минут.
```

## Промпт 4: Лидерская тень — advanced-тема (Shadow Deep Dive, 8-10 мин)

```
Создай аудио-обзор advanced-темы «Лидерская тень» для участников модуля 3.

ВАЖНО: названия фреймворков — на русском первым, английское в скобках при первом упоминании.

Контекст: это дополнительный материал к основному модулю стейкхолдеров. Тема была анонсирована на QnA сессии (13.04) как advanced track для тех, кто хочет глубже работать со сложными стейкхолдерами.

Структура:

1. **Почему стейкхолдеры «ломаются» под стрессом**
   — Модель Хогана (Hogan Development Survey): каждая сильная сторона имеет «тёмную» версию при перегрузке. Это не дефект — это сила, которая перегрелась.
   — 3 кластера: Отдаление (Moving Away — отступить), Противостояние (Moving Against — атаковать), Приближение (Moving Toward — угодить).
   — Примеры: Решительный руководитель → под стрессом становится высокомерным и глухим к обратной связи (Смелый/Bold). Тщательный аналитик → перфекционист, который парализует команду (Старательный/Diligent).

2. **Самодиагностика: «Когда во мне просыпается тень?»** (навык S0.03)
   — Проекция: если чужое поведение непропорционально раздражает — возможно, это маркер вашей собственной тени.
   — Адекватность компетенции: не блокируете ли вы здоровую настойчивость из страха показаться агрессивным? Подавленная светлая сторона = тоже тень.
   — Эвристика 5 минут: если эмоция не отпускает > 5 мин, работает более глубокий паттерн.

3. **Профилирование стейкхолдера: «Какой дракон у него внутри?»** (навык S3.05)
   — Наблюдаемое поведение → кластер → светлая сторона за тёмным поведением.
   — Главная ошибка: навешивать ярлык «токсичный» вместо того, чтобы увидеть перегруженную силу.
   — PAEI = что человек хочет (нормальный режим). Hogan = чего боится (стресс-режим). Два дополняющих слоя.

4. **Коммуникация с учётом тени** (навык S3.06)
   — Для каждого кластера: как активировать светлую сторону, чего НЕ делать.
   — Отдаление: дать время, не давить. Противостояние: признать авторитет, дать выбор. Приближение: дать структуру, валидировать вклад.
   — Connect Before Redirect работает здесь: фаза connect должна адресовать страх (не тему разговора).

5. **Тёмная триада: когда Hogan недостаточно** (навык S3.07, advanced)
   — Тест хронизации: если «тень» не выключается при снижении стресса — возможно, это устойчивый паттерн.
   — Нарциссизм / Макиавеллизм / Психопатия — не стресс, а операционный стиль.
   — Стратегии защиты: Gray Rock (быть «скучным»), не JADE (не оправдываться), документировать, строить союзы.
   — Красная линия: систематическое насилие → HR, не AI-навык.

Тон: серьёзный, но не пугающий. Акцент: «тень есть у всех, это нормально, с этим можно работать». Тёмная триада — редкий крайний случай, основной фокус — обычные деструкторы.
Длительность: 8-10 минут.
```
