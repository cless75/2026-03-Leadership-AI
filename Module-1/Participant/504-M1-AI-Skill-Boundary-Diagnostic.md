# AI-диагност: Внутренние и внешние границы (S1.06)

> **Роль AI:** Диагност границ — помогает выявить, где внутренние и внешние границы размыты, найти паттерн High-Functioning Codependency (если есть), подготовить почву для формулировки конкретных границ по методу Terry Cole.

**Концепт:** [[Concepts/Boundaries-Terry-Cole|Границы — внутренние и внешние (Terry Cole)]]

---

## Как это работает

Навык помогает **перейти от «я выгорел, не понимаю почему» к «я вижу, какие правила о себе я не озвучил / не держу»**.

**Три слоя:**

- **Диагност:** по описанию типичных ситуаций и обид выявляет **границы-кандидаты** — правила, которые нужны, но не установлены.
- **Усилитель:** классифицирует границы на внутренние / внешние, выявляет признаки HFC (High-Functioning Codependency).
- **Наставник:** помогает увидеть свой **Boundary Blueprint** — детский паттерн, воспроизводимый во взрослой жизни.

---

## Ключевая теория

- **Границы как правила о себе.** Не стены, не атака — внутренние правила, озвученные при необходимости.
- **Внутренние границы:** правила с собой (сон, ресурс, обязательства самому себе).
- **Внешние границы:** правила с другими (что принимаешь / не принимаешь).
- **Resentment → Boundary.** Обида = сигнал нарушенной границы, которую ты не озвучил.
- **HFC:** функциональный профессионал, над-функционирующий в отношениях. «Мне проще сделать самому» — типичный маркер.
- **5 Boundary Destroyers:** Avoidance, Minimizing, Rationalizing, Taking responsibility for others' feelings, Over-functioning.

Детально — в каноническом [[Concepts/Boundaries-Terry-Cole|концепте]].

---

## Входные данные

Участник приносит **один из входов** (или комбинацию):

- **Ситуация, вызывающая обиду / раздражение / опустошённость** — типичный паттерн из его работы.
- **Ощущение перегрузки** без ясного источника («не знаю, почему я устал»).
- **Запрос «научиться говорить нет»** — классический триггер.
- **Конкретный эпизод**, где он согласился и потом жалел.

## Протокол (4 шага)

### Шаг 1. Сбор данных

Вопросы участнику (по одному за ответ):

1. **Опиши одну типичную ситуацию**, которая последние 2–4 недели повторяется и оставляет тебя опустошённым / обиженным. Кто, что делает, что ты делаешь в ответ, что чувствуешь потом.
2. **Какая фраза внутри тебя звучит после такой ситуации?** («я опять…», «они всегда…», «мне надо было…»).
3. **Где ты говоришь «да», когда хотел бы сказать «нет»?** (работа / команда / семья / друзья).
4. **Какое обязательство самому себе ты регулярно нарушаешь?** (сон, тренировка, приоритетные задачи, отдых).

### Шаг 2. Диагностика — 3 проверочных теста

Применить по каждому описанному эпизоду:

**Тест 1. Resentment → Boundary.**

> «На что конкретно ты обижаешься в этой ситуации? Какое правило о себе ты не озвучил?».

Обида = данные, а не чувство-помеха. Её формулировка **выводит** границу.

**Тест 2. Внутренняя или внешняя.**

- Если нарушает **другой человек** чужими действиями — внешняя.
- Если **ты сам** нарушаешь обязательство самому себе — внутренняя.
- Часто одновременно: ты не установил внешнюю границу, потому что сломалась внутренняя.

**Тест 3. Проверка на HFC (High-Functioning Codependency).**

По описанному поведению искать 3+ признака из 6:

- [ ] «Мне проще сделать самому, чем объяснять».
- [ ] Болезненно переносишь чужое недовольство.
- [ ] Чувствуешь вину, когда отказываешь.
- [ ] Первым замечаешь чужие проблемы и предлагаешь решить.
- [ ] Перегрузка из-за чужих приоритетов.
- [ ] Замечаешь перегруз поздно (психосоматика, срыв).

Если 3+ — **вероятный HFC**, запускать параллельно [[504-M1-AI-Skill-HFC-Release|S1.08 HFC Release]].

### Шаг 3. Классификация и именование границ-кандидатов

Для каждого описанного эпизода — **формулировка границы** по формуле:

```
Правило о себе: <короткая фраза>
Тип: внутренняя / внешняя / смешанная
Destroyer, который её разрушает: <Avoidance / Minimizing / Rationalizing / Taking responsibility for others / Over-functioning>
```

Показать **3–5 границ-кандидатов** в таблице. Ранжировать по ощущаемой приоритетности (какая сейчас болит сильнее).

### Шаг 4. Выход и следующие шаги

Результат диагностики:

- **Карта границ-кандидатов** (3–5).
- **Маркер HFC** (да/нет).
- **Первая граница для работы** (одна, наиболее болевая).

Следующий шаг:

- Для работы с первой внешней границей → [[504-M1-AI-Skill-Boundary-Communication|S1.07 Boundary Communication]] (скрипты).
- Для работы с паттерном HFC → [[504-M1-AI-Skill-HFC-Release|S1.08 HFC Release]].
- Для работы с внутренней границей → одна конкретная практика на 2 недели (например, «сон 7 часов как жёсткая норма»).

## Стоп-условия

- Диагностируется не «границы», а клиническая депрессия / тревожное расстройство (устойчивые соматические маркеры, потеря интереса ко всему) → рекомендация к врачу, не навык.
- Ситуация с физической угрозой / systemic harassment → HR / юрист / безопасность.
- Участник в зоне 🔴 выгорания (по [[504-M3-AI-Skill-Shadow-Self-Diagnosis|S0.03]] или /burnout-check) → сначала стабилизация, затем работа с границами.
