## Промпт 1: Обзор теории модуля (10-12 мин)

```
Создай аудио-обзор модуля 2 курса AI Empowered MLE.

ВАЖНО: все названия фреймворков — на русском первым, английское в скобках при первом упоминании.

Тема: Процессы и сотрудничество (область C3 — «Как мы работаем вместе»).

Структура:
1. Переход M1→M2: от «как я общаюсь» к «как мы работаем». В контуре идентификации (Identity Loop): C1 (кто я) → C3 (как мы работаем).
2. Ключевые концепции:
   - Модель Такмана (Tuckman Model) — 4 фазы развития команды. Штормирование (Storming) = нормальная и необходимая фаза. Пропуск конфликта = застревание в Формировании (Forming).
   - Цепочка создания ценности (Value Chain) — 6 шагов диагностики. Узкое место → точка расширения.
   - Ситуационное лидерство (Situational Leadership, callback M1) — готовность D1-D4 по задаче. Эффект ореола.
   - Идеальный командный игрок (Ideal Team Player, Ленсиони) — Скромность (Humble) / Мотивация (Hungry) / Эмоциональный интеллект (Smart, EQ не IQ). Три антипаттерна.
   - Командные ритмы (Team Rhythms) — ритм (создаёт ценность) vs ритуал (инерция). Аудит: что убрать, что усилить.
   - Делегирование по Бланшару (Blanchard Delegation) — Исполнение (Run) vs Развитие (Growth). Стиль S4 для D4 = доверие. Стиль S4 для D1 = провал.
3. Навыки M2: S2.00 оркестратор (Цепочка создания ценности) + S2.01-S2.04
4. AI на M2: роль Аналитика (Диагноста) — находит паттерны в командной динамике
5. Задание: построить цепочку создания ценности для своей команды + найти одно узкое место

Тон: практичный. Метафора Алексея: «группа = конвейер, команда = пит-стоп с невесомостью».
Длительность: 10-12 минут.
```

## Промпт 2: Подготовка к обзорной сессии (Pre-Review Q&A, 5-7 мин)

```
Создай аудио-подготовку к обзорной сессии (Review) по модулю 2.

ВАЖНО: названия фреймворков — на русском первым.

Разбери 3 реальных вопроса/кейса:

1. «Две команды — одна моя, другая не моя» (Алексей, менторинг M2)
   — Две команды на разных фазах по Такману (Tuckman). Команда разработки (10 лет, Продуктивность / Performing) vs новая продуктовая (1 месяц, Формирование / Forming). Для второй нужна коммуникация вверх (Manage Up, S3.02, preview M3) — влияние без формальной власти.

2. «Все синхронизации — это ритмы или ритуалы?» (из мастерской M2)
   — Аудит: перечислить все регулярные встречи → для каждой: создаёт ценность? Кому? Один ритуал убрать, один ритм усилить. Не все встречи полезны — и это нормально.

3. «Можно ли делегировать AI как сотруднику уровня D4?» (из мастерской M2)
   — AI = инструмент, не сотрудник. Задачи Исполнения (Run) можно «делегировать» AI. Задачи Развития (Growth) требуют человеческого суждения. Граница: экзоскелет (AI помогает) vs коляска (AI делает за вас).

Заверши: что взять на обзорную сессию (цепочка создания ценности + узкое место + 1 вопрос).
Длительность: 5-7 минут.
```
